# Pod Silosem — Agroturystyka w Giławach

Strona wizytówka agroturystyki „Pod Silosem" w Giławach na Warmii.
Statyczna, jednostronicowa, bez frameworków (HTML + CSS + JS).

## Podgląd lokalny

```bash
python -m http.server 8080
```

Następnie otwórz <http://localhost:8080>. Można też po prostu otworzyć plik `index.html`
w przeglądarce.

## Struktura

```
index.html            — strona
assets/css/style.css  — style (paleta natury / boho)
assets/js/main.js     — menu mobilne + galeria (lightbox)
assets/img/*.webp     — zdjęcia (zoptymalizowane WebP)
```
